## Short Description

```text
Native speech-bubble callouts for co-op teammates, with "Hey, listen!", click-to-acknowledge, and an optional timer.
```

## Full Description

```bbcode
[b]Co-op Callouts[/b] adds native in-game speech bubbles for Slay the Spire 2 co-op play.

It helps highlight useful teammate moments before you commit your turn, such as setup cards, helpful effects, or other actions worth noticing.

The mod uses the game's own speech bubble style, so callouts stay lightweight and feel like part of the normal UI.

[b]Features[/b]

[list]
[*]Native Slay the Spire 2 speech bubbles.
[*]"Hey, listen!" teammate callouts.
[*]Click a bubble to acknowledge and dismiss it.
[*]Optional timer for automatic dismissal.
[*]Timeline-aware callout behavior.
[*]Vortex-ready install package.
[/list]

[b]Settings[/b]

If ModConfig is installed, Co-op Callouts can be configured in-game through the mod settings menu.

Without ModConfig, the same settings can be adjusted in the user config file created after first launch:

[code]%APPDATA%/SlayTheSpire2/CoopCallouts/config.json[/code]

Available settings:

[list]
[*][b]Enable Bubbles[/b] - Turns teammate callout bubbles on or off.
[*][b]Playable Now Only[/b] - Only shows callouts for teammate cards they can currently afford and play this turn.
[*][b]Include Support[/b] - Allows generic support callouts for helpful teammate cards that do not match a specific status keyword.
[*][b]Bubble Timer[/b] - Controls how long bubbles stay visible. Set it to 0 to keep bubbles up until clicked.
[/list]

[b]Installation[/b]

Install with Vortex, or manually extract the archive into your Slay the Spire 2 game folder.

After installation, the mod should appear here:

[code]mods/CoopCallouts/[/code]

[b]Compatibility[/b]

Co-op Callouts is made for existing co-op/multiplayer setups. It does not add multiplayer by itself.

[b]Why use it?[/b]

Co-op can get visually busy fast. This mod keeps teammate callouts readable, simple, and close to the game's own style.
```
